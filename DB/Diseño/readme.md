# Load Foundation Data Model

En esta carpeta encontrarás los siguientes archivos:

- Load Foundation Data Model.erwin: contiene el diseño de base de datos relacional para Load Foundation. Para abrirlo se requiere la versión de erwin Data Modeler 12.1 o superior.
- Load Foundation Data Model.sql: contiene los scripts generados a partir del erwin de modelo, para la creación de las tablas y demás objetos de bases de datos.
- PDM-LOAD FOUNDATION DATA MODEL.SVG: diagrama de modelo, en formato SVG. Se puede visualizar en Microsoft Edge o en visores compatibles SVG.
